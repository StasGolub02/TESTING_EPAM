Название тест кейса №1: Найти товар "кеды" испоьзуя поиск
1) Перейти на главный экран
2) Открыть поиск 
3) Написать названние товара
Ожидаемый результат: получить список товаров, которые содержат в названиие слово "Кеды"

Название тест кейса №2: Отсортировать товары по убыванию цены
1) Перейти на главный экран
2) перейти в католог
3) выбрать сортировку "Цена по убыванию"
Ожидаемый результат: товары сортируются по убыванию 
Название тест кейса №3: Отсортировать товары по Новым
1) перейти на главный экран
2) перейти в каталог
3) отсортировать товары по "Новинки"

Ожидамый результат: отобразятся товары на которые есть скидка. Отобразятся товары с пометкой "Новые"

Название тест кейса №4: Найти кеды модели OLD SKOOL
1) перейти на шлавный экран
2) перейти в каталог
3) выбрать кеды и тип oldskool
Ожидаемый результат: в окне товаров отобразятся кеды модели OLD SKOOL

Название тест кейса №5: Найти в разделе "одежда" куртки
1) перейти на главный экран
2) перейти в каталог
3) выбрать пункт одежда
4) выбоать тип "Куртки"
Ожидаемый результат: в окне товаров отобразятся только куртки

Название тест кейса №6: Удаление товаров из корзины (предусловие: в корзине есть товар)
 1) перейти в корзину
2) удалить товар
Ожидаемый результат: получение уведомление о том, что корзина пуста

Название тест кейса №7: Добавление товара в корзину не указывая размер
1) перейти на главный экран
2) зайти в каталог
3) выбрать товар типа обувь
4) нажат на кнопку "добавить в корзину"
Ожидаемый резуьтат: получить уведомление об ошибке с просьбой указать размер

Название тест кейса №8: просмотр таблицы размеров
1) перейти на главную
2) перейти в раздел контакты
3) перейти в раздел "Таблица размеров"
Ожидаемый резльтат: появление таблицы размер на странице сайта


Название тест кейса №9: Отсортировать товары по возростанию цены
1) Перейти на главный экран
2) перейти в каталог
3) выбрать сортировку "Цена по возростанию"
Ожилаемый результат: товары сортируются по возрастанию цены

Название тест кейса №10: Перейти к новинкам бренда через футер сайта
1) Перейти на главную
2) перейти в каталог 
3) в футере перейти к новинкам бренда
Ожидаемый результат: появление пользователя на странице с новинками бренда
